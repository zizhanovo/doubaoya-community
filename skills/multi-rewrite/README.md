# 多平台文案风格改写 / multi-rewrite

> 都爆鸭社区技能 · 一稿多发，本鸭帮你把一段原文铺满全平台

---

## 简介

把同一段原文一次改写成 **公众号、视频号、抖音、快手、哔站、小红书、知乎** 等平台各自习惯的表达，省去逐平台手工适配的时间。

纯本地技能：改写全程在本地完成，脚本只读取本地规则库，不联网、不上传你的文案、不需要任何 API Key。

**核心价值**

- **一稿多平台**：指定一个或多个目标平台，或输入「全部」，批量得到各平台风格成稿
- **平台调性对齐**：各版本按对应平台的标题、语气、结构与标签习惯改写，而非简单同义替换
- **清单式交付**：每个平台独立成条，分隔清晰，便于复制到各渠道发布
- **自然语言即可**：说出平台名和原文即可，无需记忆固定命令

**适用对象**

- 📱 **自媒体 / MCN** — 同一主题快速铺量到七大平台
- 🏢 **品牌 / 市场** — 统一核心信息，分平台呈现不同话术
- ✍️ **内容运营** — 把长文、活动稿、产品介绍拆成各平台可读版本

---

## 功能特性

### 核心功能

- **单平台改写**：指定如「抖音」「小红书」等，输出该平台风格全文
- **多平台批量改写**：一次选择多个平台（空格或逗号分隔），逐条列出各平台成稿
- **全平台一键改写**：输入「全部 / all / 所有」，覆盖七个支持平台
- **平台自动识别**：从话术中识别目标平台；未识别时会引导你补充平台选择
- **小红书专属标签**：小红书版本在正文末尾附带 `#` 话题标签

## 使用指南

直接说明目标平台和要改写的文案即可。

### 常用说法速查

| 意图           | 示例话术                                           | 效果                               |
| -------------- | -------------------------------------------------- | ---------------------------------- |
| 单平台改写     | 「抖音 今天天气真好，适合出门散步」                | 输出一条抖音风格成稿               |
| 多平台批量     | 「抖音 小红书 知乎 这款防晒霜很好用，清爽不油腻」  | 分别给出三个平台的改写版本         |
| 全平台一键     | 「全部 周年庆全场 5 折起，限时 3 天」              | 七个平台各一条，列表形式展示       |
| 未指定平台     | 「帮我把下面这段改写成各平台能发的：[你的原文]」   | 会询问你想覆盖哪些平台后再继续     |

### 输出示例

改写完成后，你会收到按平台分条的列表，示意如下：

```text
## 改写结果

---

### 1. 📌 公众号

[公众号风格改写内容]

---

### 2. 📌 抖音

[抖音风格改写内容]

---

### 3. 📌 小红书

[小红书风格改写内容]

#标签1 #标签2 #标签3

---
```
