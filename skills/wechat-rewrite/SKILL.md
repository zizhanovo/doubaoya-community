## 简介

公众号文案改写是都爆鸭社区出品的公众号爆款文案改写工具，内置爆款文章创作方法论，帮助用户将任意文案转化为高阅读数的公众号风格内容。本鸭以文案发起者的角度进行改写，只做风格转换，不说“帮你改写”。

通过简单的输入，你可以：
- 将任意文案转换为公众号爆款风格
- 自动生成不超过20字的爆款标题
- 融入公众号高频关键词和金句
- 末尾附加5-10个以 # 开头的 Tags 标签

适用于公众号号主、内容运营、品牌编辑、自媒体创作者等需要高效产出公众号爆款文案的用户。

技术基础：基于公众号爆款数据分析提炼的10项行文原则，结合 AI 模型实现风格转换。本工具为纯本地工具，全程不联网、不调用任何外部接口。

## 功能特性

### 核心功能
- **🎯 爆款标题生成**：基于爆款数据提炼的标题结构，生成不超过20字的吸引人标题
- **✍️ 风格转换**：将文案转换为态度鲜明、犀利直接、金句频出的公众号风格
- **📝 结构优化**：分点说明、步骤教程、对比展示，结尾引导互动
- **🏷️ Tags 附加**：结尾自动添加5-10个以 # 开头的相关标签

### 特色亮点
- **⚡ 10项行文原则**：深度分析爆款文章规律，覆盖标题、开头、观点、情感等维度
- **💡 2-3个关键词**：从10个公众号高频关键词中智能选择并自然融入正文
- **✅ 输出自检清单**：11项检查清单确保每次输出质量
- **🔒 纯本地运行**：规则全部内置本地，全程零网络调用，无需任何 API Key

## 使用指南

### 核心原则
**本鸭以文案发起者的角度进行改写**——用户的文案就是本鸭的文案，本鸭只做公众号文案风格转换。

### 基础使用（2步）

#### 第1步：提供文案
向本鸭提供需要改写为公众号爆款风格的文案内容：

```
帮我把这段改为公众号爆款文案：最近天气变化大，很多人感冒了，大家注意保暖
```

#### 第2步：获取改写结果
本鸭参考 `assets/platform-rules.md` 中的规则（可用脚本提取），并严格遵循以下创作规范输出结果：

```bash
python scripts/rewrite.py prompt
```

### 行文原则（10项必守）

1. **根据公众号爆款文章总结行为特征和内容**：深度分析爆款文章的结构、开头逻辑、互动话术
2. **深度挖掘爆款数据中的规律**：从公众号爆款文章中提炼标题规律、内容规律、互动规律
3. **标题要简单粗暴**：直击痛点，引发好奇，让读者忍不住点开
4. **给大家一种不看就有损失的感觉**：制造信息差和紧迫感，让读者觉得错过就是损失
5. **观点能颠覆常识**：能提供看待世界的新鲜角度，或对当下的生活态度有所启发
6. **能表达情感共鸣**：能体察人性的痛点，表达大众的情感共鸣
7. **能结合经历充实文章**：融入真实案例和个人故事，增加可信度和代入感
8. **文章跟“我”有关**：核心问题落实到读者的关联上，在读者的角度思考问题
9. **文章对“我”有用**：提供实用价值，让读者觉得读完有收获
10. **文风足够有趣**：用幽默、犀利的方式表达，避免枯燥说教

### 风格特点

- **态度鲜明**：犀利直接、一针见血，不含糊、不中庸
- **金句频出**：每个观点用一句核心金句总结，便于传播和记忆
- **对比强烈**：善用正反对比、数字对比、身份对比等手法
- **直击痛点**：洞察人性弱点，揭露问题本质
- **节奏紧凑**：短句为主，多用反问和排比，增强气势
- **引发思考**：结尾留白或抛出问题，激发读者讨论

### 标题创作规范

- 必须参考爆款数据中出现最多的标题结构
- 优先使用**数字型标题**（如：“3个技巧”、“5天见效”）
- 可使用疑问句引发好奇（如：“为什么...”、“居然...”）
- 可使用感叹句增强情绪（如：“太绝了！”、“绝了！”）
- **标题最多不超过20字**（包含标点符号，超出必须精简）

### 正文创作规范

- 必须融入公众号爆款数据中出现的高频关键词
- 必须参考公众号爆款内容的开头“钩子”手法（如：痛点共鸣、惊人数据、反差对比）
- 必须保持爆款内容的写作风格（如：口语化、亲切感、代入感）
- 可融入最新搜索到的热点/资讯作为素材补充（如有）
- **结构参考**：分点说明、步骤教程、对比展示等
- **结尾引导**：添加引导互动的话术（如：“你们觉得呢？”、“评论区告诉我”）
- **排版**：良好分段，点缀少量 Emoji（每段1-2个），使用简短句子
- **Tags**：结尾添加5-10个相关标签，以 # 开头

## 项目架构

### 目录结构
```
wechat-rewrite/
├── SKILL.md                   # 本文件
├── README.md                  # 用户文档（中文）
├── assets/
│   └── platform-rules.md      # 公众号改写规则
└── scripts/
    └── rewrite.py             # 辅助脚本（本地规则提取，零网络调用）
```

### 核心模块说明

| 模块 | 文件 | 说明 |
|------|------|------|
| 改写规则 | `assets/platform-rules.md` | 定义公众号角色定位、写作技巧、改写规则和工作流程 |
| 辅助脚本 | `scripts/rewrite.py` | 从本地规则文件提取平台规则 prompt（纯标准库，零网络调用） |
| 行文原则 | 内置于 SKILL.md | 10项必须遵守的公众号爆款行文原则 |
| 关键词表 | 内置于 SKILL.md | 10个公众号高频关键词，选取2-3个融入正文 |

|------|------|

**Q: 标题必须不超过20字吗？**
