# 公众号文案改写 / wechat-rewrite

> 都爆鸭社区出品 · 纯本地工具 · 零网络调用

---

## 简介

把任意草稿改写成高阅读潜力的公众号爆款文案——标题有钩子、正文有态度，并自动融入公众号高频关键词、金句与互动引导。本鸭以作者视角输出，只做风格转换。

**核心价值**

- **风格一键转化**：以作者视角输出，整体切换为公众号口语化、犀利直接的行文风格
- **爆款结构对齐**：标题控制在 20 字内，正文参考爆款开头钩子、分点结构与结尾互动引导
- **特色表达融入**：从 10 个公众号高频关键词中精选 2～3 个，自然写入正文，增强平台辨识度
- **交付即发布**：标题 + 正文 + 5～10 个 Tags 标签，拿过来就能用

**适用对象**

- ✍️ **公众号号主** — 把素材快速改写成可发布的爆款口吻文章
- 📱 **自媒体创作者** — 将笔记、演讲稿等改写成更适合微信阅读的表达
- 🏢 **品牌 / 内容运营** — 统一对外稿件的公众号风格，提升打开率与互动

---

## 功能特性

### 核心功能

- **爆款标题生成**：基于爆款数据提炼的标题结构，生成不超过 20 字的高吸引力标题
- **风格转换**：将任意文案转换为态度鲜明、犀利直接、金句频出的公众号风格
- **结构优化**：自动调整为分点说明、步骤教程、对比展示等结构，结尾引导互动
- **Tags 附加**：结尾自动添加 5～10 个以 # 开头的相关标签

### 特色亮点

- **⚡ 10 项行文原则**：深度分析爆款文章规律，覆盖标题、开头、观点、情感等维度
- **💡 精选关键词**：从 10 个公众号高频词中智能选择 2～3 个，自然融入正文
- **✅ 输出自检**：11 项检查清单确保每次输出质量稳定
- **🔒 纯本地运行**：规则全部内置本地，全程零网络调用，无需任何 API Key

## 使用指南

直接用自然语言描述需求，无需记忆命令。

### 常用说法速查

| 意图         | 示例话术                                           | 效果                               |
| ------------ | -------------------------------------------------- | ---------------------------------- |
| 改写一篇成稿 | 「帮我把下面这段改成公众号爆款风格：[你的原文]」   | 输出标题 + 正文，含互动引导与 Tags |
| 只要标题     | 「给这段内容起一个 20 字内的公众号爆款标题」       | 给出符合钩子规律的标题             |
| 强化观点     | 「改写得再犀利一点，多加金句」                     | 在爆款结构下加强态度与传播点       |
| 指定语气     | 「口语化一些，像跟读者聊天」                       | 亲切自然、短句为主的公众号口吻     |

### 提取改写规则（可选）

如需查看本鸭遵循的完整改写规则 prompt，可运行本地脚本（无任何网络请求、无第三方依赖）：

```bash
python scripts/rewrite.py prompt
```

### 输出示例

输入：*最近天气变化大，很多人感冒了，大家注意保暖*

你将收到一篇完整的公众号风格文案，包含：
- 一个不超过 20 字的爆款标题
- 态度鲜明、金句频出的正文
- 结尾的互动引导话术
- 5～10 个以 # 开头的 Tags 标签
