# 知乎文案改写 / zhihu-rewrite

> 本鸭出品 · 都爆鸭社区技能库

---

## 简介

把产品介绍、攻略分享、活动通知等原文，改写成符合知乎阅读习惯的答主式专业长文——有观点、有结构，并自然融入指南、教程、秘籍等平台特色表达。

**核心价值**

- **专业可信语感**：强调独特观点与清晰结构，贴近知乎高赞内容的表达习惯
- **特色词自然嵌入**：正文中携带 2～3 个如「指南」「教程」「案例分析」等关键词，增强平台辨识度
- **完整交付形态**：一次输出吸睛标题、口语化正文与结尾 `#` Tags
- **段落结构一致**：改写前后段落层次对齐，便于对照原文审阅
- **纯本地、零依赖**：不联网、不需要任何 API Key，仅使用 Python 标准库

**适用对象**

- 💡 **知乎创作者** — 把素材快速改成可发布的回答或专业文章口吻
- 📱 **自媒体运营** — 将其他平台内容适配知乎读者的深度阅读习惯
- 🏢 **品牌 / 市场** — 统一对外信息的专业表达与传播标签

---

## 功能特性

### 核心功能

- **知乎风格改写**：输入原文，输出带标题的知乎答主风格成稿
- **口语化表达**：用亲切、流畅的叙述拉近与读者的距离，避免生硬书面语
- **特色关键词**：从指南、教程、秘籍、经验分享、案例分析等词中选取 2～3 个融入正文
- **话题标签**：正文末尾附带与内容相关的 `#` Tags

## 使用指南

直接粘贴或描述需要改写的文案即可。如需查看完整改写规则 prompt，可运行：

```bash
python scripts/rewrite.py prompt
```

该脚本为纯本地工具，仅读取 `assets/platform-rules.md`，不发起任何网络请求。

### 常用说法速查

| 意图         | 示例话术                                                         | 效果                               |
| ------------ | ---------------------------------------------------------------- | ---------------------------------- |
| 改写成回答   | 「帮我把下面这段改成知乎风格：[你的原文]」                       | 输出标题 + 正文 + Tags             |
| 产品介绍     | 「这款耳机支持主动降噪、30 小时续航，改成知乎种草文案」          | 专业感 + 口语化，含特色关键词      |
| 活动推广     | 「周年庆全场 5 折起，改成知乎能发的活动文案」                    | 吸引注意力的标题与亲和正文         |
| 强化深度     | 「观点再鲜明一点，结构再清楚一些」                               | 在知乎风格下加强论述与可读性       |

### 输出示例

完成改写后，你将收到类似结构的内容（示意）：

```text
# 标题（有钩子、适合知乎信息流）

正文段落一：口语化展开，含「指南 / 教程」等特色词 …
正文段落二：结构清晰，观点明确 …

#数码 #耳机推荐 #深度测评
```

## 项目架构

```
zhihu-rewrite/
├── SKILL.md                   # Skill 定义与说明
├── README.md                  # 用户文档（中文，本文件）
├── assets/
│   └── platform-rules.md      # 知乎改写规则（Profile、WritingSkills、Rules、Workflow）
└── scripts/
    └── rewrite.py             # 辅助脚本（本地规则提取，无网络请求、零第三方依赖）
```
