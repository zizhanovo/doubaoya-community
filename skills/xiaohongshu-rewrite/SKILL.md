## 简介

「小红书文案改写」是都爆鸭社区里一只专啃种草文的本鸭。它把任意原文——产品介绍、使用体验、攻略分享——智能改写成符合小红书阅读习惯、高互动率的笔记成稿。

本鸭一次帮你搞定：
- 自动生成有钩子、吸睛的小红书风格标题
- 全文添加至少 5 个不重复的 emoji 表情符号，增强活力与点击欲
- 末尾自动附加以 `#` 开头、与内容强相关的话题标签
- 把书面表达改成口语化、亲切自然的小红书种草口吻，同时尽量保留原文段落结构

适用于内容创作者、品牌 / 电商运营、MCN 机构、小红书博主等需要高效产出小红书风格内容的用户。

技术基础：纯本地运行，仅使用 Python 标准库（`os`、`re`、`sys`），从 `assets/platform-rules.md` 提取规则后由 AI 模型按规则改写。**全程零网络调用，无需配置任何 API Key。**

## 功能特性

### 核心功能
- **🎯 智能标题生成**：采用二极管标题法（正面 / 负面刺激），生成有钩子、有悬念的小红书风格标题
- **😊 Emoji 润色**：全文添加至少 5 个不重复的 emoji 表情符号，标题与段落均有点缀
- **🏷️ Tags 附加**：正文末尾自动附加以 `#` 开头、与内容强相关的话题标签，便于搜索与推荐
- **📝 风格转换**：将任意文案转换为口语化、亲切自然的小红书种草表达方式

### 特色亮点
- **⚡ 即输即用**：输入原文案，直接输出标题 + 正文 + 标签的成稿，无需反复询问
- **🔒 本地优先**：脚本零网络调用、无遥测、无需 API Key，文案不出本机
- **🔧 规则可定制**：通过 `assets/platform-rules.md` 自定义 Profile / WritingSkills / Rules / Workflow 改写规则

## 一键安装

### 前置条件
- Python 3（路径：`/usr/bin/python3`）
- 无额外依赖，仅使用 Python 标准库（`os`、`re`、`sys`）

### 安装方式
将技能目录放入你的 skills 目录即可，无需任何鉴权或联网配置。

## 使用指南

### 基础使用（3步）

#### 第1步：提供文案
向 Agent 提供需要改写为小红书风格的文案内容：

```
帮我改写这篇小红书文案：这款面霜保湿效果很好，用了一周皮肤明显变水润了
```

文案可以包含产品介绍、使用体验、攻略分享等各类内容。

#### 第2步：提取改写规则
脚本从本地规则文件提取小红书 System Prompt（纯本地，不联网）：

```bash
python scripts/rewrite.py prompt
```

#### 第3步：获取改写结果
Agent 读取 `assets/platform-rules.md` 中的完整规则，严格按照小红书的 **Profile**（角色定位）、**WritingSkills**（写作技巧）、**Rules**（改写规则）、**Workflow**（工作流程）进行文案风格改写，并直接输出结果：

- 内容包含吸引人的标题
- 文中使用 emoji 表情符号（至少 5 个不重复的 emoji）
- 末尾附加 Tags（以 `#` 开头）
- 使用口语化表达拉近与读者的距离

输出完成后不再询问用户是否需要调整。

### 脚本命令速查

| 命令 | 说明 |
|------|------|
| `python scripts/rewrite.py prompt` | 输出小红书改写规则 prompt（本地提取，不联网） |

```
xiaohongshu-rewrite/
├── SKILL.md                   # 本文件
```
