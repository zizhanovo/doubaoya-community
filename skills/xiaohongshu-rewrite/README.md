# 小红书文案改写 / xiaohongshu-rewrite

> 都爆鸭社区技能 · 本地运行 · 零网络调用 · 无需 API Key

---

## 简介

把产品介绍、使用体验、攻略分享等原文，改写成符合小红书阅读习惯的正文——有吸睛标题、充足 Emoji 和结尾话题标签，读起来像真实博主在分享。这只本鸭只在你本机干活，文案不出本机。

**核心价值**

- **平台语感对齐**：口语化、有节奏，贴近小红书常见的种草与分享表达
- **视觉化阅读**：标题与正文融入 Emoji，增强活力与点击欲（至少 5 个不重复表情）
- **即改即发**：一次输出标题 + 正文 + `#` 标签，可直接用于笔记发布
- **结构保留**：在切换风格的同时，尽量保持原文段落结构，方便对照修改
- **本地优先**：脚本仅用 Python 标准库，零网络调用、无遥测、无需 API Key

**适用对象**

- 📝 **小红书博主** — 把粗糙草稿快速改成可发布的笔记文案
- 🛍️ **品牌 / 电商运营** — 统一产品卖点的小红书话术与标签
- 📱 **多平台创作者** — 将公众号、知乎等长文改写成小红书短笔记风格

---

## 功能特性

### 核心功能

- **小红书风格改写**：输入原文，输出带标题的小红书风格成稿
- **吸睛标题**：采用悬念、正反刺激等常见笔记标题手法，提升打开意愿
- **Emoji 强化表达**：全文使用至少 5 个不重复的 Emoji，标题与段落均有点缀
- **话题标签**：正文末尾附带与内容强相关的 `#` Tags，便于搜索与推荐

## 使用指南

直接粘贴或描述需要改写的文案即可。

### 常用说法速查

| 意图         | 示例话术                                                   | 效果                                   |
| ------------ | ---------------------------------------------------------- | -------------------------------------- |
| 改写成笔记   | 「帮我把下面这段改成小红书风格：[你的原文]」               | 输出标题 + 正文 + 结尾 Tags            |
| 产品种草     | 「这款精华液主打修护，改写成小红书种草文案」               | 口语化卖点表达，附相关话题标签         |
| 攻略分享     | 「周末 citywalk 路线，改成小红书攻略笔记」                 | 结构清晰、Emoji 丰富、适合笔记排版     |
| 强化标题     | 「标题再抓人一点，正文保持小红书口吻」                     | 加强开头钩子，整体仍符合平台习惯       |

### 辅助脚本（可选）

脚本仅用于本地提取改写规则 prompt，**不发起任何网络请求、无需 API Key**：

```bash
python scripts/rewrite.py prompt    # 输出小红书改写规则 prompt
```

### 输出示例

完成改写后，你将收到类似结构的内容（示意）：

```text
# 标题（含 Emoji，有吸引力）

正文段落一 ✨ …
正文段落二 💡 …
（全文至少 5 个不重复 Emoji）

#护肤 #好物分享 #种草
```
