# 情报调查员 / ai-intelligence-investigator

---

## 简介

基于 17 个搜索引擎的深度情报调查工具，自动编排搜索策略，多源交叉验证消除偏差，生成结构化调查报告。覆盖竞品分析、舆情监测、人物背景调查、信息真实性验证四大场景，遵循「多源必证、引擎适配、偏差消除」三大原则，帮助用户在信息爆炸时代高效获取可信情报。

本鸭全程在本地完成调查、比对与报告撰写——**无需任何 API Key，不依赖任何外部平台，不上传或持久化任何数据**。

**核心价值**

- **多源交叉验证**：关键信息至少 2 个独立来源确认，避免单一信源偏差。
- **智能引擎编排**：根据调查目标（中文舆情、国际视野、隐私敏感、学术验证等）自动选择最优搜索引擎组合。
- **可信度量化**：ABCD 四级信源分级 + ✅/⚠️/❌/🔍 多源确认标注，让信息可信度一目了然。
- **结构化报告**：竞品分析、舆情事件、人物背景、信息验证四类报告模板，输出专业、可复用的调查结果。
- **纯本地运行**：调查全程在本地完成，结果直接交付给用户，不涉及任何外部鉴权或数据持久化。

**适用对象**

- 🏢 **品牌方 / 企业管理者** — 竞品情报调查、市场动态追踪、合作方背景调查。
- 📊 **分析师 / 研究人员** — 多源数据收集、信息交叉验证、结构化报告生成。
- 📰 **媒体 / 内容创作者** — 热点事件溯源、舆论走向分析、信息来源核实。
- 🔒 **法务 / 风控人员** — 人物背景调查、诉讼记录排查、合作风险评估。

## 功能特性

### 核心功能

- **竞品情报调查**：多引擎搜索竞品产品功能、用户口碑、市场表现，输出 SWOT 分析及行动建议。
- **舆情事件调查**：事件还原 + 多视角收集 + 时间线重建，呈现完整舆论图谱。
- **人物背景调查**：身份信息核实、职业履历追溯、专业成就验证、风险信号排查。
- **信息交叉验证**：信息溯源 → 多源比对 → 权威验证，给出已确认 / 待确认 / 已否定的综合判定。
- **引擎自动编排**：根据调查目标智能匹配合适的搜索引擎组合（最多 17 个引擎），消除单一引擎偏差。
- **可信度分级标注**：ABCD 四级信源分级，✅已确认 / ⚠️待确认 / ❌已否定 / 🔍单一来源 四种标注状态。

### 特色亮点

- **17 引擎广度覆盖**：Baidu、Google、DuckDuckGo、WeChat、Toutiao、Brave、Bing INT 等，支持按地区、按用途灵活组合。
- **三原则驱动**：多源必证（≥2 独立来源）、引擎适配（按目标智能编排）、偏差消除（不同引擎/地区交叉比对）。
- **四级信源可信度**：A 级（官方/权威媒体）、B 级（行业媒体）、C 级（社交媒体）、D 级（匿名来源），让信息可信度可量化。
- **四类专业报告模板**：竞品情报、舆情事件、人物背景、信息验证各配备标准化输出模板，专业可复用。
- **零配置、零依赖**：开箱即用，无需注册、无需密钥、无需网络回传，调查报告直接落地本地。

## 使用指南

直接用自然语言描述调查需求，无需记忆命令或搜索引擎语法。

### 常用说法速查

| 意图 | 示例话术 | 效果 |
| ---- | -------- | ---- |
| 竞品调查 | 「帮我调查一下 Notion 的产品功能和用户口碑」 | 3 轮搜索（广域→深度→交叉验证），输出结构化竞品报告 |
| 舆情追踪 | 「帮我追踪最近 XX 事件的舆论走向」 | 事件还原 + 多视角收集 + 时间线重建 |

| 信息验证 | 「验证一下『XX 公司获得 10 亿融资』是否属实」 | 溯源 → 多源比对 → 权威验证 → 综合判定 |

**用户口碑分析**：正面/负面评价汇总，口碑关键词总结

**市场表现**：融资轮次/金额、用户规模、市场份额（标注来源与可信度 A~D 级）

## 重要数据说明

- 调查结果基于多搜索引擎交叉验证，关键信息至少 2 个独立来源确认后才标注为「已确认」。
- 信源分为 ABCD 四级：A 级（官方/权威媒体）、B 级（行业媒体/专业平台）、C 级（社交媒体/自媒体）、D 级（匿名/未验证来源）。
- 搜索结果受搜索引擎索引时效影响，实时性要求高的调查建议使用时间过滤（如 `tbs=qdr:d`）。
- 调查模式自动适配：中文舆情优先 Baidu + WeChat + Toutiao，国际视野优先 Google + Brave + Yahoo，隐私敏感优先 DuckDuckGo + Startpage。
- 所有调查结果均来源于公开可访问的搜索引擎，不涉及非公开数据采集。
- 本技能纯本地运行，不需要任何 API Key，不向任何外部服务上传或持久化调查数据。
